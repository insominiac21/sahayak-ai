# Scheme 1 — PMAY-U 2.0 (Pradhan Mantri Awas Yojana – Urban 2.0)

## Overview
PMAY-U 2.0 is a “Housing for All” mission for **urban areas**, providing central assistance so eligible households can **construct, purchase, or rent** an affordable pucca house through States/UTs, ULBs, and partner lending institutions.

## Eligibility
- State:
  - All participating **States/UTs** (implementation is via States/UTs/ULBs/PLIs; coverage depends on notified towns/ULBs).
- Age:
  - Not explicitly defined as a number in the scheme guidelines. Beneficiary is defined as a **family** (husband, wife, unmarried children). Applicant should have Aadhaar/Aadhaar Virtual ID.
- Income:
  - **EWS:** up to ₹3 lakh/year  
  - **LIG:** ₹3 lakh to ₹6 lakh/year  
  - **MIG:** ₹6 lakh to ₹9 lakh/year  
- Category:
  - EWS / LIG / MIG (as above). Preference is given to groups like widows, single women, persons with disabilities, senior citizens, transgenders, SC/ST, minorities, and other vulnerable sections (subject to verification by local authorities).
- Occupation:
  - Not mandatory as a strict eligibility field, but special focus is mentioned for groups like Safai Karmi, street vendors, artisans (e.g., PM Vishwakarma), Anganwadi workers, construction workers, slum/chawl residents, etc.

## Checklist
(Exact list varies by State/ULB + the vertical you apply under. This is the minimum “always mentioned” set.)
- Aadhaar / Aadhaar Virtual ID for all eligible beneficiaries (and self-attested Aadhaar copy where required)
- Declaration / undertaking that:
  - family does **not** own a pucca house anywhere in India
  - applicant has **not** received a house under Central/State/Local housing schemes in the last 20 years
- For BLC (Beneficiary Led Construction): proof of land ownership / clear title (as asked by implementing agency/ULB)
- Basic application details needed for verification: household category (EWS/LIG/MIG), income details, contact details
